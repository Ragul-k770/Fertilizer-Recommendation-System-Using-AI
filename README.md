## How to Run the Project
1. Install Java (JDK 8 or above)
2. Import the project into NetBeans / Eclipse
3. Configure MySQL database using fertilizer.sql
4. Run Main.java file

## Dataset
The dataset contains soil parameters such as Nitrogen, Phosphorus, Potassium, pH value, and crop type.

## Output
The system displays the recommended fertilizer based on the given inputs.
