# Source Code

This folder contains the complete Java implementation of the **Fertilizer Recommendation System Using AI**.

## Files Description

- **Main.java**  
  Entry point of the application. It collects soil and crop details and displays the recommended fertilizer.

- **SoilData.java**  
  Represents soil parameters such as Nitrogen, Phosphorus, Potassium, pH value, and crop type.

- **FertilizerRecommendation.java**  
  Contains the logic to recommend suitable fertilizer based on soil nutrient levels.

## Technology Used
- Java
- Rule-Based Decision Logic

## Purpose
To help farmers select the most appropriate fertilizer using soil data and basic AI logic.
